畿内诸侯，封地在京畿内，在周天子那有职务。在王权稳定的时候，非常容易利用在中央的官职谋求利益。
就像现在中央部委的高官在地方办事有很多便利。

当天子的威信下降的时候，地方诸侯就更有势力。就像各省不再给中央上税，中央也没有权利调拨地方的资源，那地方的一把手才是实力派。

郑国本是畿内诸侯。郑桓公，脑筋灵活，未雨绸缪。利用自己中央的职务在东方给自己规划了一块新开发区。其实西周初期，周武王就意识到要“宅兹中国”，即要在疆域中心地带建立国都。武王做了规划，成王平定三监之乱后营建了雒邑。郑桓公的策略延续了这个思路。成功地从畿内诸侯转变成幾外诸侯。

母親為弟弟請的第一個封地是制。 
地理位置在洛陽以東，今河南省滎陽市市區西北部16公里的汜水鎮境內。為洛陽東邊門戶和重要的關隘，因相傳西周穆王在此圈養猛虎而得名虎牢關。南連嵩岳，北瀕黃河，山嶺交錯，自成天險。有「一夫當關，萬夫莫開」之勢，為歷代兵家必爭之地。 
鄭國是西周后期才產生的一個諸侯國。始封的國君是鄭桓公姬友。鄭桓公是周王朝第10位天子周厲王的少子，也是第11位天子周宣王的弟弟。周宣王在位的第二十二年，鄭桓公姬友初封於鄭。這個鄭是在今陕西华县的东方。位置在鎬京的東邊，是一個畿內諸侯。並非後來春秋時期的鄭國。 
西周末期，周室衰微，鄭桓公積極謀尋全身之策，不愿與周室共存亡。鄭桓公聽從史伯建議，利用自己王朝大臣的身份，通過向虢、鄶之君賄賂，取得十邑之地作為新鄭國之在東方的立足點。把原先封地的民衆往東遷徙。從畿內諸侯轉變成畿外諸侯。 
鄭武公在位的二年（公元前769年），鄭武公滅亡鄶國，佔領鄶國都城制邑。自己占據制以東的地盤，把以西的地盤給了周平王。《左傳》在魯莊公二十一年（前673年）有言：「鄭伯平周王室之亂，與之武公之略，自虎牢以東。」 
制邑即虎牢關一帶，地處鄭國最北端，北依黃河 ，岸如絕壁，控此一地可制鄭國全境；另外，虎牢關又是東方諸侯通向周王朝首都洛邑的必經之地，是鄭國商業發展的重要關口 
可以說制這座城是鄭國與周天子領地和西方諸國的重要門戶。鄭武公花了大力氣去擴建此城，甚至不惜破例給被打成奴隸身份的商朝遺民平反，讓他們參與修建。 
鄭莊公對這制邑的重要性堅信不疑。這中認知是鄭國三代人實際走出來，從戰場上用鮮血和生命得到的。 
一旦制邑不掌握在自己手中，自己的王位和整個鄭國都處在危險之中。 

後世的史實也證明了這一點。《左傳》在襄公二年記載：秋七月，晉師侵鄭。會於戚，謀鄭故也。孟獻子曰：「請城虎牢以逼鄭。」 
晉軍侵略鄭國，仲孫蔑和晉國荀罃、宋國華元、衛國孫林父、曹國人、邾國人在戚地會見，這是為了商討征服鄭國的辦法。仲孫蔑建議：“請進駐虎牢關逼迫鄭國。” 
後來采取的方式就是在“冬，復會於戚，齊崔武子及滕、薛、小邾之大夫皆會，知武子之言故也。遂城虎牢，鄭人乃成。” 
也就是，冬季，再次在戚地會見，齊國的崔武子和滕國、薛國、小邾國的大夫都參加會見，這是由於知武子這一番話的緣故。於是就在虎牢加强工事。在虎牢關被控制后，鄭國人便同意講和。 
當然這是發生在莊公之後的事情。莊公能從歷史和當時的地緣政治上認知到這一點，對一個十四嵗的年輕人相當不容易。 
鄭武公在位時期，高舉“尊王”與“愛民”兩面大旗。對外方面是“尊王”，就是尊重周天子的地位。對内“愛民”，安撫從被自己滅亡的鄶、東虢以及其他八個附庸國的人民，着力發展經濟。 
“尊王”的一個具體策略就是讓出制邑（即虎牢關）以西的一大片土地給周天子。自己牢牢占據制邑。可以以方便勤王為藉口，實現監視周朝和西方各國之實。 
對於鄭莊公執政初期的首要任務是延續爺爺和父親鄭桓公和武公的方針 - 維護周天子的地位，發展自己的勢力。三年不改父志，有利於穩定自己的位置。控制制邑是重要一環，絕不可放棄。 

京城在春秋時期屬鄭國，曾為鄭國東遷後的第一個國都。鄭武公消滅鄶國后，將鄭國都城由京城“遷到溱洧（今河南新鄭一帶）”。京城是舊都城。京城的建制原本就是按照國都修建的。 
朝中的大夫也看到了這一點。祭仲曰：“都城過百雉，國之害也。先王之制：大都不過參國之一，中五之一，小九之一。今京不度，非制也，君將不堪。” 
翻譯成白話文就是。大夫祭仲說：“分封的都城如果城墻超過三百方丈長，那就會成為國家的禍害。先王的制度規定，國內最大的城邑不能超過國都的三分之一，中等的不得超過它的五分之一，小的不能超過它的九分之一。京邑的城墻不合法度，非法制所許，恐怕對您有所不利。” 

《左傳·莊公二十八年》對國度有解釋：“凡邑有宗廟先君之主曰都，無曰邑。邑曰築，都曰城。”。有宗廟和先君神主的叫做“都”，沒有的叫做“邑”。建造邑叫做“築”，建造都叫做“城”。 
國君居於國都，主持宗廟囯之大事。其餘的城邑必須逐次降級。大中小的城邑分別是國度的三分之一，五分之一，和九分之一。不能超過這個限制。 
二百多年后，孔夫子在魯國主政時，主導堕三都，就是堕毁三桓（魯國公族季孫氏、叔孫氏、孟孫氏）三都私邑。以增强君權，減少内耗。當時魯國三桓分魯軍的權利，各家家臣又對三桓各懷異心。甚至發生家臣起兵反鲁的事情。對，不是三桓要反魯國國君，是三桓的家臣起兵反鲁，率军攻入鲁国国都曲阜，鲁定公和三桓大夫，也就是季桓子、孟懿子和叔孙武叔，躲在季氏之宫，武子之台。三桓和鲁定公同爲魯桓公的后代。也是説魯國的國君和同姓的掌權大夫差點被人家做成全家桶，打包送走。 
京城是舊都城，原本就是按照國都修建的。但，已經不再是國都，應該是京邑。 
《左傳》的原文：請京，使居之，謂之京城大叔。就是段在得到京這個城池后，沒有主動把它降爲一個邑，仍保持了“城”的規模，稱其爲京城，旁人看得明白，於是把段稱爲“京城大叔”。 
擺出了要與鄭莊公分立的架勢。這背後應該有母親武姜的支持。 
面對大夫祭仲的提醒，出自十四歲少年君主之口的那句「多行不義必自斃」，更像是一句自我安慰的話。他是多麽希望母親能提醒弟弟段遵循禮制。 
接下來京城大叔的操作恰恰和鄭莊公的希望背道而馳。 
“既而大叔命西鄙北鄙貳于己。” “大叔又收貳以為己邑，至于廩延。” 
貳在《左傳》中均不是數字的二，是懷有貳心，違背、叛離的意思。 
大叔先是讓西鄙的北鄙兩個地方同時聽命于國君和自己。「鄙」是春秋時期邊邑的稱呼，《說文解字》云：「五酇為鄙，百家為酇」，因此一鄙約為五百家，另據《周禮》記載，都鄙是王公子弟、卿大夫的采地，既然是采地就存在耕地、糧倉等重要物資，這為擴充軍隊和建設城防起到了重要作用。 
隨後，大叔直接把西鄙和北鄙改為自己統轄的地方，一直擴展到廩延。廩延即今日的延津，是古黃河邊上的重要的渡口，跨過黃河向北即到達衛國。 
這相當於北京派了一個副國級幹部去上海，結果他讓江蘇和浙江直接給他上稅，而不聽命與中央。或者臺北總統府委任一名大員到臺南，大員讓高雄和嘉義歸自己管。 
北京不忍，總統府也坐不住。

美国科罗拉多州丹佛市樱桃溪学院(CCI: Cherry Creek Institution) 是一家专注于IT(Information Technology)技术的职业培训学校，其使命是培训在美的中国同胞，经过四个月到半年左右时间的学习，能够找到并胜任十万美金或者更高年薪的IT专业工作。

众所周知，IT行业是一个处处都有高薪工作的高科技领域。它的学习门槛并不高，只要有正常的逻辑思维能力的高中生，经过一段时间的学习，均可在这个行业中找到自己的位置，拿到不错的薪水。

自2025年2月始，CCI启动PostgreSQL数据库管理员培训项目，招收学员学习著名开源数据库PostgreSQL(简称PG)的相关技术，目标是找到并胜任PG数据库管理员和开发相关的专业工作岗位。PG数据库发展势头良好，很多工作的薪水达到15万到20万美金的年薪，而且很多工作明确规定可以在家里远程上班。

本培训项目的详情如下：

- 招生对象：高中文凭以上，具备在美国合法工作许可的中国同胞。
- 培训费用： 共计12,000美元， 分为首款5,000$和尾款7,000$。 其中首款需在入学之前缴纳，学员在找到工作后，自上班之日起，半年之内，付清尾款7,000$。
- ## 就业零风险： 自缴纳首款之日起，如果学员在一年内没有找到IT专业方面的工作， CCI无理由退还首款5000$。彪悍的培训质量不需要解释。
- H1B签证：优秀的国际留学生经过我校培训后找到工作，我们可以为其办理H1B签证。
- 培训形式：现场教学、网上互动教学和视频教学相结合。

![学习流程](flow.svg)

我们知道，学习是有规律的。刚开始入行一个行业时，一无所知，所以学习进度比较缓慢，为此，CCI采用成本最高但是效果最好的现场培训模式。经过一个月的现场学习后，学员已经入门了，就可以回家，在家里继续学习。外地来丹佛参加现场培训的学员，CCI将协助其住宿，但食宿费用需要学员自理。

网络学习主要采用Zoom等流行的网络会议软件，教师通过分享屏幕的方式进行互动式教学。这个过程持续一个月，学员拥有 了独立自主学习的能力，就进入了第三个阶段，即CCI制作高质量的培训视频供学员继续深入学习。

学员把规定的教学内容学习完毕后，即可申请毕业考试。考试通过的学员，CCI将为其提供免费且专业级的简历修改和就业指导服务，该服务直至学员拿到工作为止。CCI同时和业界公司进行合作，如果学员被CCI的合作伙伴录取，自上班第一天始，工作满一年后，CCI将无条件退还尾款7000$。

CCI拥有十余年的IT职业培训经验，顺利就业的学员数百位。CCI同学会中的毕业生遍布全美各地，为新毕业的学员提供广泛的人脉支持。让学员扎扎实实学到真技术，顺顺利利拿到好工作，稳稳当当捧住金饭碗，和和美美过上小日子，是我校的办学宗旨。
